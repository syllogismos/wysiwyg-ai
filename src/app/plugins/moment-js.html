Moment.js
Parse, validate, manipulate, and display dates in JavaScript.
Format dates
Use the moment().format() method to format dates.
Method Result
moment().format('MMMM Do YYYY, h:mm:ss a')
moment().format('dddd')
moment().format("MMM Do YY")
moment().format()
Relative time
Use the fromNow(), the startOf() and the endOf() methods for relative time formatting.
Method Result
moment("20111031", "YYYYMMDD").fromNow()
moment("20120620", "YYYYMMDD").fromNow()
moment().startOf('day').fromNow()
moment().endOf('day').fromNow()
moment().startOf('hour').fromNow()
Calendar time
Use the subtract() and the add() methods with the calendar() to add and subtract dates.
Method Result
moment().subtract(10, 'days').calendar()
moment().subtract(2, 'months').format()
moment().subtract(15, 'days').startOf('day').fromNow()
moment().add(10, 'days').calendar()
moment().add(2, 'months').format()
moment().add(15, 'days').startOf('day').fromNow()
