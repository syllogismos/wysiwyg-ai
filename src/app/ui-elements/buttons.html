Default buttons
Use the .btn-{{'{color}'}} class for default buttons. Use the .btn-icon class for buttons with icons
Outlined buttons
Add the .btn-outline class for outlined buttons
Rounded buttons
Use the .btn-rounded class for rounded buttons
Outlined rounded buttons
Combine the .btn-rounded with the .btn-outline class for outlined rounded buttons
Flat buttons
Use the .btn-flat for flat buttons
Circle buttons
Use the .btn-circle for circular buttons. Add the .btn-xs, .btn-sm or the .btn-lg for additional sizes
Default button group
Use any of the .btn-{{'{color}'}} classes inside the .btn-group class for default button groups
Rounded button group
Use any of the .btn-{{'{color}'}} classes inside the .btn-group .btn-group-rounded class for rounded button groups
Outline button group
Use any of the .btn-{{'{color}'}} classes inside the .btn-group .btn-group-outline class for outline button groups
Outline rounded button group
Use any of the .btn-{{'{color}'}} classes inside the .btn-group .btn-group-rounded .btn-group-outline class for outline rounded button groups
Flat button group
Use any of the .btn-{{'{color}'}} classes inside the .btn-group .btn-group-flat class for flat button groups
