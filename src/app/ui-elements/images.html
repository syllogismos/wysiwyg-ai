Rounded images
For rounded images use the .rounded class

Thumbnail images
For thumbnail images use the .img-thumbnail class

Circular images
For circular images use the .rounded-circle class

Zooming animations
Apply the .zoom-in-1, .zoom-in-2, .zoom-out-1 or the .zoom-out-2 classes for zooming image animations
Sliding animations
Apply the .slide-left, .slide-right, .slide-up or the .slide-down classes for sliding image animations
Rotating animations
Apply the .rotate-{{'{1-4}'}} classes for rotating image animations
Opacity animations
Apply the .opacity-1 or the .opacity-2 classes and combine it with any of the bg-{{'{color}'}} helper classes for opacity image animations
Other effects
Additional effects that work only on the latest standards compliant browsers
