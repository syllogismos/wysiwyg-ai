Naming conventions
File and folder name definitions and conventions used in the template

Every component included in the template follows the {{'{controller}'}}/{{'{view}'}} naming convention.

For example, for the ui-elements/tooltips demo url, located at url: http://delta-angular.batchthemes.com/ui-elements/tooltips:

  1. .html files are located in the src/app/ui-elements folder
  2. .scss files are located in the src/app/styles/ui-elements folder
  3. .ts files are located in the src/app/ui-elements folder

  4. For this example, the file locations are the following:

    1. src/app/ui-elements/tooltips.html
    2. src/app/styles/ui-elements/tooltips.scss
    3. src/app/ui-elements/tooltips.ts
Folders
Important folders in the template

  1. src/app

    Includes all the typescript and html files used in the template.

  2. src/styles

    Includes all the .scss files used in the template.

  3. src/scripts

    Includes all external javascript and typescript files that were not installed using bower or npm.

  4. src/assets

    Includes all static assets used in the template. This includes images, .json and .svg files.

  5. bower_components

    Includes all the dependencies installed with bower.

  6. node_modules

    Includes all node packages, components and dependencies installed with NPM.

Files
Important files in the template

  1. src/index.html

    Template home page

  2. src/app/app.module.ts

    Module definitions and routes are defined here.

  3. src/app/app.component.ts. src/app/app.component.html

    Typescript and html initialization files files. Layouts, colors and components shown are defined here.

  4. src/app/services/colors.ts

    Global colors and palettes are defined here

  5. src/app/styles/_variables.scss

    Global .scss variables are imported and defined here

  6. src/app/styles/_colors.scss, src/app/styles/_palettes.scss and src/app/styles/palettes/*.scss

    Colors and palettes are defined in this files and folder

  7. src/app/styles/layouts/*.scss

    Template layouts are defined in this folder

  8. src/app/styles/_mixins.scss

    .scss mixins are defined here

  9. src/app/styles/helpers/*.scss

    Includes global .scss helpers
