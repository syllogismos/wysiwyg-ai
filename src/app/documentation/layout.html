Layout
Frequently asked questions

  1. How do I change the page layout?

    Every page has a data-layout attribute in the <body> tag. If you want to change the layout you need to change this attribute. You can do it in the html files or you can do it using javascript. There are 6 layout options:

    • default-sidebar-1
    • collapsed-sidebar-1
    • top-navigation-1
    • top-navigation-2
    • off-canvas-1
    • sidebar-over-1

    For demo purposes, the layout is also set with javascript in the scripts/app.js file in the following lines: 

    $('body').attr('data-background', config.background);
$('body').attr('data-navbar', config.navbar);
$('body').attr('data-sidebar', config.sidebar);
$('body').attr('data-top-navigation', config.topNavigation);
$('html').attr('class', 'loading')

    Make sure you remove or comment this lines when you build your files for production.

    Also the index.html and the views inside the pages folders use the following layouts:

    • empty-view-1

    The style for each layout is defined in the src/app/styles/layouts folder using the following naming convention:

    For example, for the default-sidebar-1, the .scss file is located at src/app/styles/layouts/default-sidebar-1.scss and will get compiled to css/layouts/default-sidebar-1.css. The colors for this layout are defined in the src/app/styles/palettes folder.

  2. How can I modify the layout elements widths and heights?

    Layout variables are defined in the src/app/styles/_variables.scss file. Here you can modify the sidebar width, navbar height, top navigation height and other global variables.

  3. How do I change the navbar?

    The navbar content is defined inside the <nav> element with the .navbar-1 class. The layout and position of the navbar for each layout is defined in the src/app/styles/layouts/*.scss files and the height of the .navbar-1 element is defined in the src/app/styles/_variables.scss file

  4. How do I change the top navigation elements?

    The top navigation elements are defined inside the .top-navigation class. The layout and position of the top navigation elements for each layout is defined in the src/app/styles/layouts/*.scss files and the height of the .top-navigation-1 and .top-navigation-2 elements is defined in the src/app/styles/_variables.scss file

  5. How do I change the left sidebar?

    The left sidebar is defined inside the .left-sidebar class. The layout and position of the left sidebar for each layout is defined in the src/app/styles/layouts/*.scss files and the width of the .left-sidebar-1 element is defined in the src/app/styles/_variables.scss file
