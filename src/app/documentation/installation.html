Delta
Installation instructions

Delta is an advanced, responsive dashboard UI kit built using Bootstrap 4, the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web, Angular 4 and jQuery, the fast, small, and feature-rich JavaScript library. It includes 5 different layouts, 5 background color styles, more than 100 HTML files in every layout and lots of widgets and custom made reusable components to help you develop your next application.

Delta is 100% ready to use. Just unzip the files you just bought and do the following:

First go the angular folder and run npm install -g @angular/cli to install the Angular CLI package.

Then run npm install to install all the required dependencies.

After you install everything, then run npm start and open your browser at http://localhost:4200.

To build a production ready bundle, run the ng build -prod command.

Then and go to the dist folder where your production ready files are generated. The files in the dist folder are ready to use and can be uploaded to any server.

The template includes the main structural elements:

  • Navbar
  • Top navigation
  • Jumbotron
  • Left sidebar
  • Right sidebar
  • Main content

The template comes with 4 different color schemes and includes the following layout options:

  • Default sidebar
  • Collapsed sidebar
  • Off canvas navigation
  • Sidebar over layout navigation
  • Top navigation 1 and Top navigation 2
  • Full page empty views: empty-view-1
