Frequently asked questions

  1. Where can I find the colors available in the template?

    Colors are defined in the src/app/styles/_colors.scss and in the src/services/colors.ts files. Default color palettes are defined in the src/app/styles/_palettes.scss file and additional palettes can be found in the src/app/styles/palettes folder

  2. How do I change the page background and text colors?

    Background colors are defined for each palette in the src/app/styles/_colors.scss file. Palettes are defined in the src/app/styles/_palettes.scss file and in the src/app/styles/palettes folder. After you define your palette colors in this file, change the data-background attribute in the <body> element to match the name of your palette.

  3. How do I change the navbar, left sidebar, right sidebar, top navigation and jumbotron colors?

    Navbar, left sidebar, right sidebar, top navigation and jumbotron colors are defined for each palette in the src/app/styles/palettes folder. After you define your palette colors in this files, change the data-navbar or the data-sidebar or the data-top-navigation attributes in the <body> element to match the name of your palette.

  4. How do I change the colors of other elements, such as tables and dropdowns?

    Default palettes are defined in the src/app/styles/_palettes.scss file. For other elements, colors and palettes are defined in the src/app/styles/palettes folder. If there is a palette defined you will find it in this folder. If there are not palettes defined for an element, then the default palette styles are applied.
