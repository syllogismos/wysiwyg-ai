Angular CLI
Use the following Angular CLI tasks and commands for development. For more information go to

For detailed instructions on how to install or update the @angular/cli project, please go to this url: https://github.com/angular/angular-cli. After you install the @angular/cli project, then run npm install to install all the required dependencies needed to run the template.

Important commands
Use the following commands for common tasks.

ng help

Use this to view all the available options the ng command has available

ng start

Use this task to start the development server for your application

ng build -prod

Use this task to bundle your application into a production ready folder
