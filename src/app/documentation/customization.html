1. How do I open the template?

    First, make sure the @angular/cli package is installed. For more information go to this url.
    Then you need to run npm install to install the template following by the npm start command to start the development server.

  2. Where can I find the starter packages?

    The starter packages are located in the angular-starter-project folder. This folder includes a sample application that is ready to use and customize. It includes all the basic components included with the template with all the additional elements and widgets removed. You should use this to start developing your project.

  3. How do I install/uninstall a dependency with Bower?

    Run bower install <dependency-name> --save. After this then you should add the installed dependencies to the .html files.
