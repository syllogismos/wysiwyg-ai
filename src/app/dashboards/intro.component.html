Escherboard

Click below to checkout sample neural networks built on the dashboard.

Here you can build deep learning models by adding layers and connecting them. Currently we support any directed acyclic graph formed with the layers available.

Instructions

  • You can add layers to the canvas by clicking on the respective layer buttons
  • Click on a layer to select it, or you can select multiple layers by click and drag and making a rectangle.
  • Select layers to copy, paste, drag, delete.
  • You can pan the canvas by right clicking anywhere on the canvas and dragging.
  • You can link layers by clicking on the link layers button, and then clicking on the input layer and then clicking on the output layer. The order in which you click matters here, as we are trying to build a directed acyclic graph.
  • You can save the network you built by clicking on the "Save to DB", Don't forget to name your network.

Notes

  • You can create the dataset in the datasets page, it basically a http link of the data
  • Specify the loss function, optimizer
  • You can specify the hyperparameters by basically giving a comma seperated values when applicable, and we will launch all the variants of the experiment based on the hyperparameters
  • While the training is happening, you can see layer level gradient statistics at each epoch of each hyperparamenter variant of the experiment in the experiment page.
  • You can monitor machine level cpu/load/memory stats to see and modify if your compute resources are being used effectively

Datasets

Notes

  • You can create a dataset by giving us any downloadable http link.
  • While training we download the data to the respective machines automatically and train.
  • Currently we only support image datasets.
  • The upload should be a zipped file with the following directory structure.
