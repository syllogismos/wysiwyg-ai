Animation delays using time
Apply the delay-{{'{millisecond}'}} classes in 100ms increments to any individual element that has animations applied to it for complete control of the animation delay
Fast animation delays
Use the .delay-fast class on the parent element of the animation for faster animation timings
Normal animation delays
Use the .delay-normal class on the parent element of the animation for normal animation timings
Slow animation delays
Use the .delay-slow class on the parent element of the animation for slower animation timings
